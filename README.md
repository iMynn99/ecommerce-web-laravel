**E-Commerce Web Application**

A web-based e-commerce application for UMKM Afify Store that allows customers to browse products, place orders, and complete payments online.

**Tech Stack**

1. PHP (Laravel)
2. MySQL
3. REST API
4. Midtrans Payment Gateway
5. Ngrok (payment testing on local environment)

--**Features**--
**User**

1. Product listing & detail
2. Shopping cart
3. Checkout & online payment
4. Order history

**Admin**
1. Product management (CRUD)
2. Order management
3. Payment status monitoring
4. Admin dashboard
